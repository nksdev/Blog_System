# 🚀 Blog System

A lightweight and developer-friendly **Blog System with CMS and GitHub automation**, built for speed, simplicity, and full control.

---

## 📌 Features

- 📝 Static blog system (HTML + JSON based)
- ⚙️ Custom CMS for managing posts
- 🔄 GitHub sync automation scripts
- 🚀 Easy setup & start scripts
- 📂 Clean and organized structure
- 🧠 Developer-focused workflow

---

## 📁 Project Structure

```

blog_system/
│── blog/                # Static blog frontend
│   ├── index.html
│   ├── posts.json
│   └── images/
│
│── cms/                 # Blog CMS system
│   ├── blog_cms.py
│   └── requirements.txt
│
│── scripts/             # Automation scripts
│   ├── create_github_repo.sh
│   └── sync_to_github.sh
│
│── outputs/             # Logs / outputs
│── setup.sh             # Initial setup
│── start.sh             # Run project
│── INSTALLATION_GUIDE.md
│── QUICKSTART.md
│── todo.md

````

---

## ⚡ Quick Start

### 1️⃣ Clone Repository

```bash
git clone https://github.com/nksdev/blog_system.git
cd blog_system
````

---

### 2️⃣ Setup

```bash
chmod +x setup.sh
./setup.sh
```

---

### 3️⃣ Start Project

```bash
chmod +x start.sh
./start.sh
```

---

## 🧠 CMS Usage

Run CMS manually:

```bash
cd cms
pip install -r requirements.txt
python blog_cms.py
```

👉 Use CMS to:

* Add new blog posts
* Edit posts
* Manage blog content

---

## 🔄 GitHub Automation

### Create Repository

```bash
./scripts/create_github_repo.sh
```

### Sync Blog

```bash
./scripts/sync_to_github.sh
```
